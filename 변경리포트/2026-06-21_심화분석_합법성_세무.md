# 심화 분석 — 합법성·적용범위·사업자 대응(세무 포함) · 2026-06-21

> **주제:** ① 문신은 완전히 합법인가 ② 반영구화장도 문신에 포함되는가 ③ 사업자는 어떻게 해야 하는가
> **조사 범위:** 문신사법 제정문(공포·시행·부칙), 대법원 전원합의체 판결, 공중위생관리법, 국세청 업종코드·부가세 자료
> **판정:** 🟢 심화 자료 추가 — 기존 모니터링에 "법적 지위 정의 + 세무/사업자 실무"를 보강

---

## 1. 문신은 이제 "완전히" 합법인가?

**한 줄 답: 형사처벌은 사라졌다(비범죄화). 하지만 "완전 합법"보다는 "2027-10-29부터 면허제로 전환"이 정확하다.**

| 구분 | 현재(2026~2027.10) | 2027-10-29 이후 |
|------|---------------------|------------------|
| 비의료인 시술 형사처벌 | ❌ 처벌 안 됨 (대법 전합 2026-05-21 전원일치) | 면허 있으면 합법 |
| 면허·등록 의무 | 아직 없음(제도 미시행) | **면허·업소등록 필수** |
| 무면허·미등록 영업 | 형사 리스크 거의 없음(과도기) | **문신사법 위반(위법)** |
| 위생·소비자 피해 | 민사·행정 책임 상존 | 동일 + 행정처분 강화 |

- **근거 판례:** 대법원 전원합의체 2026-05-21 **전원일치**로 "문신 시술은 질병의 예방·치료 목적 의료행위로 보기 어렵다"며 1992년 판례를 34년 만에 변경 → 현행 의료법 기준으로도 비의료인 문신은 처벌 대상 아님.
- **근거 법령:** 문신사법(법률 제21070호) **공포 2025-10-28 / 시행 2027-10-29**(공포 후 2년 경과). 면허·국가시험·업소등록 의무는 **시행일부터** 효력.
- **요지:** 지금은 "면허 없이도 처벌 안 되는 과도기 합법", 2027-10-29부터는 "면허 있어야 합법".

## 2. 반영구화장도 "문신"에 포함되는가?

**한 줄 답: 예. 반영구화장 = 미용문신 = 문신사법 적용 대상. 별도 면허가 아니라 단일 면허로 통합될 가능성이 높다.**

- 문신사법은 문신을 **① 서화문신(타투·예술)** + **② 미용문신(반영구화장 — 눈썹·입술·헤어라인·두피문신)** 으로 분류 → **동일 법률로 규율**.
- 통합 이유: 둘 다 **바늘로 피부에 색소를 주입하는 침습 시술**이며 장비·염료 차이가 크지 않음 → **단일 "문신업(문신사)" 체계**. (유형별 지원 차등은 향후 가능)
- 판례 일치: 대법원이 **눈썹 문신·헤어라인(반영구화장)** 비의료인 시술도 무죄 확정(2026-06) → 서화문신과 동일하게 비범죄.
- **두피문신** 도 미용문신 범주 → 동일 적용.

> 실무 시사점: 반영구·문신 종사자는 "타투가 아니다"가 아니라 **문신사법 면허·등록 대상자**로 보고 준비해야 함.

## 3. 사업자는 무엇을, 어떻게 준비해야 하나 (세무 포함)

### (A) 지금 당장 — 과도기(2026~2027.10)

| 항목 | 내용 |
|------|------|
| **사업자등록** | 소득 발생 시 필수. 미신고 영업은 세무 리스크. |
| **업종코드** | 관행상 **「피부미용업」 930923** 또는 기타 미용 계열. 정확한 코드는 홈택스 → 기준·단순경비율 조회 후 관할 세무서 확인. |
| **부가가치세** | 미용 용역은 **과세**(의료기관 면세와 다름). 일반과세 또는 간이과세 신고. 연 공급대가 4,800만 원 미만 간이과세자는 부가세 납부의무 면제 구간. |
| **종합소득세·현금영수증** | 미용업은 **현금영수증 의무발행 업종** → 누락 시 가산세. |
| **위생·기록** | 시행 전이라도 감염관리·시술기록(일자/염료/부위)·동의서 체계화 → 민사 리스크↓ + 향후 등록 시 유리. |

### (B) 2027-10-29 시행 대비 — 합법 영업 4단계

1. **국가시험 합격 → 보건복지부 장관 면허 취득** (2027년 말 첫 시험, 사례형·실무형. 예시 문항 2026년 말 공개)
2. **문신업소 개설등록** — 시설·장비 요건 갖춰 **시·군·구청에 등록**(등록제)
3. **임시등록 특례(핵심)** — 일정 요건자는 **시행 후 2년 되는 날까지 임시 개설등록** 가능. 단 **그 2년 안에 면허 취득 후 다시 정식 개설등록** 필요 → 기존 종사자 영업 공백 없이 전환
4. **염료·마취제 합법 공급** — 복지부 **문신용 일반약 신규 허가** 추진 → 불법 경로 의존 정리

### (C) 신설 가능성(모니터링 대상)
- 문신업 **별도 사업자등록 업종코드 신설** + **세제 감면** 검토
- **중소벤처기업부 시설비용 지원**(정책자금·저금리 융자) 검토 → 발표 즉시 활용 권장

## 4. 기대 효과 & 예방책 요약

**기대 효과**
- 과도기에도 형사 리스크 없이 영업 가능 + 세무·위생 체계를 미리 갖추면 2027 면허·등록 전환이 매끄러움.
- 단일 면허라 반영구 전업자도 동일 시험으로 진입 → 자격 시장 선점 기회.

**예방책(리스크)**

| 리스크 | 예방책 |
|--------|--------|
| 세무 누락(현금영수증·부가세) | 업종코드·과세유형 확정, 현금영수증 의무발행 준수, 세무사 상담 |
| 2027 면허 미준비 | 2026년 말 예시 문항 공개 즉시 학습 시작, 국가시험 일정 확보 |
| 임시등록 시기 놓침 | 시행(2027-10) 전후 시·군·구청 임시등록 공고 모니터링 → 즉시 신청 |
| 시설기준 미달 | 2026년 7월 시설기준안 공개 즉시 갭 분석, 개조·장비 견적 확보 |
| 염료·마취제 불법 공급 | 복지부 허가 제품·합법 공급경로로 전환 |
